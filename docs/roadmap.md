# Monthly Mail Club - Roadmap

## Phase 0 - Validation

Objectives:

* Interview 20-30 creators
* Validate workflow pain
* Validate willingness to pay
* Identify common workflows

Success:

* At least 10 creators express interest
* At least 5 indicate willingness to pay

---

## Phase 1 - Foundation

Features:

* Next.js application
* Better Auth
* Prisma
* Postgres
* Artist accounts
* Platform plans

Success:

* Artist can create account
* Artist can access dashboard

---

## Phase 2 - Club Management

Features:

* Clubs
* Membership plans
* Club items
* Plan item assignments

Success:

* Artist can define club structure

---

## Phase 3 - Subscriber Management

Features:

* Subscriber CRUD
* CSV import
* Subscription management
* Subscriber status tracking

Success:

* Artist can manage subscribers without spreadsheets

---

## Phase 4 - Address Collection

Features:

* Address portal
* Secure tokens
* Address validation
* Address updates

Success:

* Subscriber can self-manage address

---

## Phase 5 - Monthly Mail Runs

Features:

* Generate mail runs
* Fulfillment calculations
* Required item counts
* Mail run locking

Success:

* Artist knows exactly what to prepare

---

## Phase 6 - Fulfillment

Features:

* Packing status
* Shipment status
* Fulfillment dashboard

Success:

* Artist can track fulfillment progress

---

## Phase 7 - Label Generation

Features:

* Avery 5160 export
* Avery 8160 export
* Thermal label export
* CSV export

Success:

* Artist can print labels directly

## Phase 8 - Stripe Connect

Features:

* Stripe onboarding
* Membership billing
* Webhooks
* Customer portal

Success:

* Artist can accept subscriptions directly

## Phase 9 - Integrations

Features:

* Patreon import

Success:
